# ASRS SmartView L504 - Goodyear 🏭

Dashboard moderno para el monitoreo de tiempos de paro (Downtime) y confiabilidad del área **ASRS (Automated Storage and Retrieval System)** en la planta Goodyear. Este proyecto reemplaza la interfaz antigua por una solución de alto rendimiento, optimizada para dispositivos móviles y alineada con los activos maestros de SAP.

---

## 🚀 Propósito del Proyecto
Optimizar la visualización de indicadores críticos (KPIs) para el área de mantenimiento y procesos, permitiendo identificar rápidamente las causas de las averías en los equipos **L504** (Cranes, Robots, Conveyors y Horseshoes).

## ✨ Características Principales
* **Visualización Avanzada:** Gráficos dinámicos interactivos mediante ApexCharts.
* **Integración SAP:** Filtros de activos organizados jerárquicamente según la estructura de planta.
* **Detalle de Averías:** Tabla integrada con descripción de fallas, tiempos de paro y órdenes de trabajo.
* **Diseño Responsive:** Totalmente compatible con celulares y tablets para uso en terreno (piso de planta).
* **Modo Debug:** Panel de diagnóstico técnico para verificar la conexión con el servidor local.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript (ES6+).
* **Gráficos:** ApexCharts API.
* **Iconografía:** Bootstrap Icons.
* **Datos:** Conexión vía Fetch API a servidores MySQL internos (Intranet Goodyear).
